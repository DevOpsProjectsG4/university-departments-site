Courses

The Computer Science Department offers a comprehensive curriculum designed to support undergraduate and graduate degree programs. The courses listed below provide theoretical foundations, practical skills, and exposure to modern computing technologies.

Degree Programs Overview

BS Computer Science

A four-year undergraduate program focusing on programming, algorithms, software systems, and emerging technologies.

MS Computer Science

A graduate program emphasizing advanced computing concepts, research, and specialization in high-demand areas.

PhD Computer Science

A research-intensive doctoral program aimed at producing original contributions to the field of computer science.

Undergraduate Curriculum Structure

The BS Computer Science curriculum is structured across four academic years, with increasing depth and specialization.

Year 1 – Foundations

Programming Fundamentals

Introduction to programming, problem-solving, and algorithmic thinking using modern programming languages.

Discrete Mathematics

Covers logic, sets, relations, graphs, and mathematical reasoning essential for computer science.

Computer Systems

Basic understanding of computer organization, hardware components, and operating system principles.

Year 2 – Core Computing

Data Structures & Algorithms

Study of efficient data organization, algorithm design, and performance analysis.

Object-Oriented Programming

Design and implementation of modular and reusable software using object-oriented principles.

Database Systems

Database design, normalization, SQL, and introduction to NoSQL databases.

Year 3 – Advanced Topics

Operating Systems

Process scheduling, memory management, file systems, and concurrency.

Software Engineering

Software development lifecycle, testing methodologies, and project management.

Computer Networks

Network architectures, protocols, data transmission, and security fundamentals.

Year 4 – Capstone & Electives

Final Year Project

A major capstone project involving problem analysis, system design, implementation, and documentation.

Artificial Intelligence

Introduction to intelligent systems, search algorithms, and basic machine learning techniques.

Cybersecurity

Fundamentals of information security, cryptography, and secure system design.

Graduate-Level Courses

Graduate courses focus on advanced theory, research methodologies, and specialization areas.

Machine Learning

Supervised and unsupervised learning, neural networks, and real-world applications.

Advanced Software Engineering

Large-scale system design, cloud architectures, and DevOps practices.

Research Methodology

Research design, academic writing, and experimental evaluation techniques.

Skills & Learning Outcomes

Strong analytical and problem-solving skills
Proficiency in multiple programming paradigms
Experience with databases and system design
Preparation for industry and research careers
